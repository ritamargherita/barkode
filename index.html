Barkode MVP

Read the label before it reaches the bowl.

A first product prototype for scanning dog and cat food and checking what the label does and does not prove under current European rules.

EU labelling rules FEDIAF 2025 guidance Honest about missing data

What this version can do

  • Flag label fields that should be present
  • Distinguish complete vs complementary feed
  • Check mineral-feed edge cases
  • Calculate calcium-to-phosphorus ratio when available

This MVP avoids fake certainty. Nutrient adequacy checks that need manufacturer data are shown as “needs more data”.

Regulation (EC) No 767/2009 defines complete and complementary feed and lists key mandatory label elements for pet food.
